## Introduction

### Project Description

1. **TIL** 프로젝트는 매일 새롭게 배운 내용과 특정 상황에서 발생한 이슈를 정리하기 위한 레포입니다.
2. 현재는 **HTTP**, **JPA**, **Git**에 대한 내용을 정리하는 과정에 있으며 **해당 개념 하나하나를 정확히 숙지하는 목적**이 있습니다.

### Goal to Achieve
    
1. **작은 부분**이 모여 **빙산**이 되듯이 하루하루 부족한 부분을 채워나가고자 하는 목표가 있습니다.
2. **단순히 기록**하는 것에 초점을 맞추기 포다는 배운 내용을 **정확히 숙지**한다는 목표가 있습니다.
